
Description
------------------------------------
 1) Designed the classes for implementing a chess game. 
 2) Implemented moves for every piece
 3) Added functionality to store move history and provide an ability to restore the state 
	of the board to the 'nth' move in the game, including both players' moves
 4) Interactive UI. The player with active turn is indicating to play his turn by a flashing message against his name
 5) Has the ability to Restart or Quit the game
 6) Displays a History/State tracker for every move.


 Assumptions
------------------------------------
 1) If the King of the other player, gets replaced by any player's move, that player wins the game
 2) Check-mate condition not implemented
 3) Not played under any time contraints
